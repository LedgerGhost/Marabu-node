# LedgerGhost's Marabu Node

A Marabu node. See the [Marabu Protocol Description](https://marabu.com/protocol).

# Installing and running

This client is designed to run with `bun` 1.3.10 or higher,
because it uses Stage 3 decorators.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
